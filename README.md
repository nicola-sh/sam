# SAM — System Admin Management

Единое десктоп-приложение (Python, PyQt6):

| Вкладка | Назначение |
|---------|------------|
| **Скачивание** | Логи с кластеров/серверов по SSH (read-only) |
| **Обезличивание** | Поиск PAN/IP/паролей, маскирование локальных файлов |
| **Подключения** | Vault, кластеры, микросервисы |
| **Пользователи** | Учётки SAM (bcrypt) |

## Запуск

```bat
python -m sam
```

Первый запуск — мастер vault + admin.

## Данные

Всё в `sam_app_data/`:

| Путь | Содержимое |
|------|------------|
| `config.yaml` | Настройки SAM |
| `vault.enc`, `master.key` | SSH, инфраструктура |
| `users.json` | Пользователи |
| `regcon/` | Настройки и вывод обезличивания |
| `audit/`, `logs/` | Аудит и тех. лог |

## Тесты

```bat
python -m pytest tests/ -q
```

## Код

Только пакет **`sam/`** (включая подмодуль `sam.regcon` для вкладки обезличивания).

Офлайн-библиотеки: [`docs/AVAILABLE_LIBRARIES.md`](docs/AVAILABLE_LIBRARIES.md).
