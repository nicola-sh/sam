# SAM

Десктоп: **Скачивание** логов (SSH) + **Обезличивание** (RegCon) + vault и пользователи.

## Безопасность

| Что | Как |
|-----|-----|
| Пользователи SAM | `users.json`, bcrypt |
| SSH, IP, инфраструктура | `vault.enc` (AES-GCM), master-пароль или `master.key` |
| RegCon | Только локальные файлы, без сети |
| Аудит | `sam_app_data/audit/*.jsonl` |

## Запуск

`python -m sam` — vault → вход SAM → главное окно.

## Вкладки

1. **Скачивание** — кластер/сервер, микросервисы, период, grep, ZIP.  
2. **Обезличивание** — RegCon (загружается при первом открытии вкладки).  
3. **Подключения** — vault, FO/BO/ATM, микросервисы.  
4. **Пользователи** (admin).

`SAM_MASTER_KEY` — авто-разблокировка vault (base64, 32 байта).
