# SAM

## Структура

| Путь | Назначение |
|------|------------|
| `sam/ui/` | Главное окно, вкладки |
| `sam/regcon/` | Обезличивание (вкладка «Обезличивание») |
| `sam/vault/` | Шифрование SSH и инфраструктуры |
| `sam/services/` | Выгрузка логов по SSH |

## Безопасность

- Пользователи: bcrypt в `users.json`
- Серверы: `vault.enc` (master-пароль или `master.key`)
- RegCon: только локальные файлы, данные в `sam_app_data/regcon/`

## Запуск

`python -m sam`

`SAM_MASTER_KEY` — авто-разблокировка vault (base64, 32 байта).
